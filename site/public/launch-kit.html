One-line summary

InvoiceGen is a free, open-source, local-first macOS invoicing app with a Rust CLI for private, scriptable invoice workflows.

Short description

InvoiceGen is a local-first invoice generator for macOS. Freelancers can manage clients, projects, payment details, invoice line items, taxes, notes, backups, and PDF-ready exports without creating a cloud account. The native Swift app and Rust CLI share the same local store, making billing workflows private, scriptable, and open source.

Launch description

InvoiceGen is a native macOS invoicing workspace for freelancers and small teams who want invoice data to stay on their Mac. It supports clients, projects, payment details, line items, due dates, taxes, notes, backups, and PDF-ready invoice exports. Unlike browser-only invoice generators or hosted accounting tools, InvoiceGen does not require a signup, subscription, cloud backend, or telemetry service. The app ships with a Rust CLI that reads the same local store, so users and AI agents can inspect, create, list, render, export, and restore invoices through scripts.

I built InvoiceGen for freelancers and small teams who want invoices to stay local instead of living in a hosted accounting app. It is a native macOS app plus a Rust CLI that shares the same local store, so invoices can be created, listed, rendered, exported, backed up, and restored from scripts. The app is free and Apache-2.0. I am looking for feedback on the local data model, CLI ergonomics, and which invoice workflows should be added next.
